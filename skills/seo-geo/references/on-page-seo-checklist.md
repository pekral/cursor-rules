# On-Page SEO Checklist

## Title Tag

- [ ] Primary keyword included
- [ ] Brand name present
- [ ] Secondary keyword where natural
- [ ] Length: 50-60 characters recommended

## Meta Description

- [ ] ~150-160 characters
- [ ] Compelling and action-oriented
- [ ] Aligned with query intent
- [ ] Contains primary keyword naturally

## Open Graph and Twitter Cards

- [ ] `og:title` aligned with page title
- [ ] `og:description` aligned with meta description
- [ ] `og:image` set to a 1200x630 image when available
- [ ] Twitter Card tags present (`twitter:card`, `twitter:title`, etc.)

## Content Structure

- [ ] Primary keyword in `H1`
- [ ] Descriptive `alt` attributes on images
- [ ] Internal links to related content
- [ ] External links with `rel="noopener noreferrer"` when appropriate
- [ ] Public pages are indexable (no accidental `noindex`)
- [ ] Reasonable page load time

## Canonical and Indexing

- [ ] `<link rel="canonical">` points to the correct URL
- [ ] No duplicate content issues between pages
- [ ] Pagination handled with proper `rel="prev"` / `rel="next"` or canonical
